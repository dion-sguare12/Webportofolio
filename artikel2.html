Saya masih ingat betul saat pertama kali mencoba membuat website. Waktu itu, saya hanya bermodalkan laptop dan rasa penasaran. Awalnya saya bingung harus mulai dari mana, tapi setelah mencari tutorial di YouTube dan membaca beberapa artikel, saya mulai mengerti tentang HTML dan CSS.

Prosesnya Tidak Mudah

Salah satu tantangan terbesarnya adalah ketika saya mencoba membuat layout yang rapi. Saya berkali-kali mengubah kode karena tampilannya berantakan. Tapi dari situlah saya banyak belajar tentang struktur kode, debugging, dan pentingnya kesabaran dalam belajar hal baru.

Hal yang Saya Pelajari
  • Pentingnya merencanakan desain terlebih dahulu.
  • Cara menggunakan elemen HTML secara benar.
  • Mengatur tampilan dengan CSS, terutama padding, margin, dan warna.

Sekarang saya merasa lebih percaya diri saat membuat website. Website pertama saya mungkin masih sederhana, tapi saya bangga karena saya buat sendiri dari nol. Semoga pengalaman saya bisa memotivasi kamu yang sedang memulai!
